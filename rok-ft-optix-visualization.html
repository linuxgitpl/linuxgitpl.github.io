Project Structure and Demo

You can go step by step, or see results.

  1. prepare: CreateEmptyVersion
  2. configure the communication
  3. import tags from ethernet
  4. import tags from OPC-UA
  5. do graphic work with templates
  6. link data
  7. do simulation: check results and alarms

Features

  • get project from Git repository
  • Ethernet IP Tag Importer: import online tags and types
  • OPC-UA Tag Importer: import online tags and types
  • emulator: run on simulator with Optix runtime
  • connect database: view logs with tables and charts
  • edit controls: edit recipes, track historical data
